# Compliance checklist

## Nabídka před publikací

- [ ] Primární oficiální zdroj je dostupný.
- [ ] Výše a typ odměny odpovídají zdroji.
- [ ] Jsou uvedeny minimální platby/vklad, termíny a omezení.
- [ ] Jsou uvedeny relevantní poplatky a investiční rizika.
- [ ] `lastVerified` a `nextReview` jsou vyplněné.
- [ ] Affiliate odkaz je označen a existuje čistý oficiální zdroj.
- [ ] Nejsou použity falešné recenze, ratingy, počty uživatelů ani naléhavost.
- [ ] Editor zkontroloval text proti zdroji.

## Web

- [x] Neplatné detailní URL nezobrazuje jinou nabídku.
- [x] Nedoložené recenze a výherci odstraněni.
- [x] Placeholder analytiky odstraněn.
- [x] Neověřené nabídky nejsou v aktivním přehledu ani sitemapě.
- [x] Affiliate disclosure je u CTA.
- [ ] Právně ověřená identita provozovatele a povinné údaje.
- [ ] Samostatné zásady soukromí a cookies.
- [ ] Právně ověřená pravidla soutěží a doporučovacích odměn.
- [ ] Souhlasy/licence pro používaná loga a značky.

## Investiční obsah

- [x] Robinhood obsahuje upozornění na volatilitu kryptoměn.
- [x] Není prezentována garantovaná pevná hodnota proměnlivé odměny.
- [ ] Právní kontrola investičního disclaimeru před produkcí.
